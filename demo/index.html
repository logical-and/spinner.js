spinner.js

A jQuery plugin that allows you to easily generate highly customizable loading animations.


Examples

The easiest way to use spinner.js is like this: 

<-- Include jquery and spinner.min.js -->
<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="spinner.min.js"></script>

<div class="wave"></div>
<div class="domino"></div>

<script type="text/javascript">
  $('.wave').spinner('wave');
  $('.domino').spinner('domino');
</script>

Configuration options can be passed as the second argument: 

$('body').spinner('wave', {
  'bars':    '7',
  'height':  '24px',
  'width':   '15px',
  'spacing': '1px',
  'speed':   '92',
  'opacity': '1',
  'color':   '#383b3d'
});
